TinyTask
========
<br>

Truly Minimalist Task Automation
--------------------------------
The files in this folder are for the [TinyTask program][TinyTask].
<br>

Why is it here?
---------------
This is arguably not a keyboard layout program at all, but a macro recorder in its own right. 
I just found TinyTask so incredibly ... well, tiny. And well-made at that. Make of it what you will. 

For more info, see its page. And if you somehow distrust the .exe enclosed in this folder, download it there. 
<br>

TinyTips
--------
Some tips and observations:
- EPKL Send doesn't play well with this (state mappings). Suspend/quit EPKL for best results.
- **Shift+Ctrl+Alt+R**: Start/stop recording
- **Shift+Ctrl+Alt+P**: Start/stop playback
- If these hotkeys don't work anymore, try closing TinyTask by clicking the X button on its window.
